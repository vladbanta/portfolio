# 2D Aircraft Game

## Description
The objective of the game is to control an aircraft, avoid obstacles, destroy enemies, and collect power-ups to increase the score. The game features a scrolling background, collision detection, and various interactive elements like enemies, asteroids, crates, and power-ups.

---

## Features
- **Player Control:** Navigate the aircraft using arrow keys and rotate using **W, A, S, D**.
- **Shooting Mechanics:** Press **Space** to fire bullets.
- **Enemy Interaction:** Avoid asteroids and destroy enemy planes.
- **Power-ups:** Collect crates for bonus points and syringes for extra lives.
- **Scrolling Background:** Simulates flight movement.
- **Collision Detection:** Real-time collision handling for bullets, enemies, and power-ups.
- **Save & Load:** Save the game state with **F4** and load with **F5**.

---

## Technologies Used
- **Programming Language:** C++
- **Graphics Library:** OpenGL
- **Window Management:** WinAPI
- **Data Structures:** Vectors, Sprites, Linked Lists
